# mallofneeds

A rewrite of [Mall of Needs](https://github.com/reiswindy/Mall-of-Needs) in Crystal using CrSFML.

## Installation

Run from project root:

```bash
$ shards --production build --release
```

## Usage

Run from project root:

```bash
$ bin/mallofneeds
```
